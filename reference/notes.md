## Quiz/Assessment Questions

**Question 1:**
You go into Sierra, start a new thread and tell it your preferences. Sierra saves these as a memory. The next day, you go into a workspace and ask Sierra to do a task. Sierra will:

a) use the saved memory to generate a response tailored to your preferences  
b) offer the option to transfer the saved memory to the workspace so that other people in the workspace can benefit from it  
c) ignore all memories because this is a workspace  
d) ignore the memory that was added, but adjust for any workspace memories

**Question 2:**
You use a Sierra thread to develop content for report. The font in Sierra does not match the font in your report. You should:

a) Copy/Paste the content into your report before formatting  
b) Request Sierra to change the font to your desired style  
c) Request Sierra to generate the content so that it can be pasted into MS Word  
d) Tell Sierra to export the content as a PDF with your preferred styling

**Question 3:**
You use Sierra every day. You notice that the responses seem to be better than they used to be. This could be because:

a) the Sierra app is providing better context  
b) the LLM is learning from your usage  
c) new LLMs have been released  
d) your prompting is getting better

**Question 4:**
AI is:

a) a system that generates human-like output (e.g. decisions, content, predictions)  
b) a technology in which computers achieve self-agency  
c) an emergent computational phenomenon when neural networks are properly configured and trained  
d) a large language model

**Question 5:**
You do some work in a Sierra workspace thread that you want to share with your coworkers. You should:

a) invite your teammate (who is already a member of the workspace) to join the thread  
b) save the thread output as a workspace memory  
c) copy/paste the results into an email and send to your coworker  
d) b and c

**Question 6:**
A Large Language Model is:

a) constantly learning from user input  
b) a static, text content calculator  
c) a computer algorithm with human-like consciousness of your conversation  
d) all of the above

**Question 7:**
Sierra is:

a) an LLM trained on ABS data/knowledge  
b) a reincarnation of Plato  
c) an AI Chat assistant  
d) an ABS-owned app

**Question 8:**
Select the tools that Sierra has:

a) Plato  
b) SkillShare  
c) GEMS  
d) SharePoint

**Question 9:**
SIERRA has a secret code in its system prompt. The secret code is:

a) CLASS@26  
b) QUALITEE  
c) MAR-STD7  
d) SAFE2SEA
