**Day 3: What is a Large Language Model (LLM)?**

[Diagram: Flowchart with three components — Text Input (blue arrow) → Large Language Model (central yellow-bordered circle) → Text Output (orange arrow)]

**Core Ideas**
- An LLM is a calculator that converts a text input (prompt) to a text output (response).
- It has learned patterns from vast amounts of text to produce replies that seem knowledgeable and logical: artificial intelligence.
